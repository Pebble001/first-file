# Premier League 2023/24 Dashboard

An interactive Power BI dashboard analysing the 2023/24 English Premier League season — tracking team performance across goals, points, wins, losses, and draws.

---

## Overview

This project explores team-level performance data across the full Premier League 2023/24 season. The dashboard is designed for fans, analysts, and stakeholders who want a quick visual summary of how each club performed.

**Business Question:** Which teams dominated the 2023/24 season and what does the points/goals breakdown look like across the league?

---

## Tools Used
- **Power BI** — dashboard design and interactive visuals
- **Excel / CSV** — raw data source
- **Power Query** — data loading and quality checks

---

## Dashboard Features

| Visual | Description |
|---|---|
| Summary Cards | Total wins, losses, draws, goals scored, and points |
| Bar Charts | Wins, draws, losses, and points compared across all 20 teams |
| Data Table | Full league table with team stats |
| Team Slicer | Filter all visuals by a specific club |

---

## Key Insights
- Summary cards give an at-a-glance view of the full season standings
- Bar charts make it easy to compare clubs head-to-head on any metric
- The interactive slicer allows deep dives into individual team performance

## How to Reproduce
1. Download the CSV dataset
2. Open Power BI Desktop
3. Load the CSV via **Get Data → Text/CSV**
4. Apply the transformations in Power Query
5. Rebuild visuals or open the `.pbix` file directly
